============================================================
  RK Web Monitor v2.0.0 — Быстрый старт
============================================================

ЧТО ЭТО:
  Современная панель мониторинга продаж R-Keeper 7.
  Заменяет устаревший WebMonitor 4.11 (ASP.NET + dojo mobile).

ПОРТ ПО УМОЛЧАНИЮ:
  http://localhost:8083
  (как в оригинальном WebMonitor — обратная совместимость)

СЛУЖБА WINDOWS:
  Приложение устанавливается как Windows Service "RKWebMonitor"
  через NSSM (Non-Sucking Service Manager).
  - Автозапуск при старте Windows (опционально)
  - Автоматический перезапуск при сбое
  - Запуск от имени SYSTEM (без необходимости входа пользователя)

ПЕРВЫЙ ВХОД:
  Логин: admin
  Пароль: admin
  ⚠️ СМЕНИТЕ ПАРОЛЬ В НАСТРОЙКАХ ПОСЛЕ ПЕРВОГО ВХОДА!

БЫСТРЫЙ СТАРТ:
  1. После установки служба запустится автоматически
  2. Откройте браузер: http://localhost:8083
  3. Войдите как admin/admin
  4. Нажмите кнопку "Настройки" (шестерёнка в правом верхнем углу)
  5. Настройте подключение к вашему серверу R-Keeper 7:
     - Тип: RK7 TCP (для прямого подключения)
     - Адрес: 192.168.1.10:15551 (IP:порт вашего сервера RK7)
     - Логин/пароль: учётка менеджера RK7
     - CryptKey: при необходимости
  6. Нажмите "Проверить соединение" (иконка Wi-Fi)
  7. Создайте пользователей в соответствующей вкладке

УПРАВЛЕНИЕ СЛУЖБОЙ:
  Пуск → RK Web Monitor:
    - Открыть в браузере → http://localhost:8083
    - Запустить службу → start.bat
    - Остановить службу → stop.bat
    - Перезапустить службу → restart.bat (для применения новых настроек)
    - Удалить программу

  Через командную строку (от администратора):
    sc start RKWebMonitor     - запуск
    sc stop RKWebMonitor      - остановка
    sc query RKWebMonitor     - статус
    sc config RKWebMonitor start= auto  - автозапуск

СМЕНА ПОРТА:
  1. Зайдите в Настройки → вкладка "Система"
  2. Измените значение поля "Порт" (по умолчанию 8083)
  3. Нажмите "Сохранить настройки"
  4. Нажмите "Перезапустить службу" (или запустите restart.bat)
  5. После перезапуска приложение будет доступно на новом порту
  6. Если меняли порт — обновите правило Windows Firewall

ФАЙЛЫ И ПАПКИ:
  bin\nssm.exe         — менеджер служб
  bin\watchdog.js      — обёртка для корректного перезапуска службы
  start.bat            — запуск службы
  stop.bat             — остановка службы
  restart.bat          — перезапуск службы
  RKWebMonitor.bat     — главный запускатель
  app\                 — код приложения (Next.js standalone)
  data\                — база данных SQLite (rkwebmon.db) + restart.flag
  logs\                — логи сервера (server.log, ротация 10 МБ)
  node\                — встроенный Node.js

БЭКАП:
  Для бэкапа скопируйте файл data\rkwebmon.db
  В нём хранятся все настройки серверов, пользователи, история сообщений
  и системные параметры (включая порт).
