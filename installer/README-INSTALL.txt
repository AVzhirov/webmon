============================================================
  RK Web Monitor v2.0.0 — Быстрый старт
============================================================

ЧТО ЭТО:
  Современная панель мониторинга продаж R-Keeper 7.
  Заменяет устаревшее WebMonitor 4.11 (ASP.NET + dojo mobile).

ПОРТ ПО УМОЛЧАНИЮ:
  http://localhost:3000

ПЕРВЫЙ ВХОД:
  Логин: admin
  Пароль: admin
  ⚠️ СМЕНИТЕ ПАРОЛЬ В НАСТРОЙКАХ ПОСЛЕ ПЕРВОГО ВХОДА!

БЫСТРЫЙ СТАРТ:
  1. Запустите ярлык "RK Web Monitor" из меню Пуск или рабочий стол
  2. Откроется консоль — НЕ ЗАКРЫВАЙТЕ ЕЁ во время работы
  3. Через 5 секунд автоматически откроется браузер
  4. Войдите как admin/admin
  5. Нажмите кнопку "Настройки" (шестерёнка в правом верхнем углу)
  6. Настройте подключение к вашему серверу R-Keeper 7:
     - Тип: RK7 TCP (для прямого подключения)
     - Адрес: 192.168.1.10:15551 (IP:порт вашего сервера RK7)
     - Логин/пароль: учётка менеджера RK7
     - CryptKey: при необходимости
  7. Нажмите "Проверить соединение" (иконка Wi-Fi)
  8. Создайте пользователей в соответствующей вкладке

ОСТАНОВКА:
  - Закройте консольное окно (Ctrl+C)
  - Или запустите "Остановить сервер" из меню Пуск

ФАЙЛЫ И ПАПКИ:
  start.bat          — запуск сервера
  stop.bat           — остановка сервера
  RKWebMonitor.bat   — главный запускатель (= start.bat)
  app\               — код приложения (Next.js standalone)
  data\              — база данных SQLite (rkwebmon.db)
  logs\              — логи сервера
  node\              — встроенный Node.js
  bin\nssm.exe       — менеджер служб (опционально)

БЭКАП:
  Для бэкапа скопируйте файл data\rkwebmon.db
  В нём хранятся все настройки серверов, пользователи и история сообщений.
